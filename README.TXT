# nanocosme README.TXT
# CCC 3/2021

1. Install
cp nanocosme-dist.tar.gz <directory>
cd <directory>
tar xvvf nanocosme-dist.tar
[rm nanocosme-dist.tar]
tar xvvf nanocosme-dist.tar.gz
[rm nanocosme-dist.tar.gz]

2. Write application
<any-tex-editor> application.cpp

3. Compile application
make

4. Config application
<any-tex-editor> config.xml

5. Start
sudo ./nanocosme start [-v]

6. Stop
sudo ./nanocosme stop [-v]
